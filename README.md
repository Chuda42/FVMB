# FVMB
Simulador de sistema de reservas de hostales. Proyecto para Programación 4 Facultad de Ingenieria UdelR

## Descripción
Sistema que implementa la realidad modelada en Documentation.

## Correr el programa

Compilar:
```
make
```

Comenzar a ejecutar:
```
./main
```
Limpiar para volver a compliar:
```
make clean
```
